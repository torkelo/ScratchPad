Support

Github

The Kibana project is based out of Github. You can find the source, send me a message, file a bug report or submit a feature request at our Github Project Page »

Submit an Issue

If you think you've found a bug, you probably have. When submitting a bug or feature request please be as detail oriented as possible. If submitting a bug include all steps required to reproduce. Indicate versions for: ruby, elasticsearch, logstash, your browser. Kibana issues »

Go Fork and Conquer

Knowing something is broken sucks. Knowing something is fixed is awesome. Consider forking Kibana on Github and submitting a pull request. Patches in diff format are cool too!

Fork me on Github »
